## Can use data structures such as MinPriorityQueue directly in leetcode without any import
For example, 
    const pq = new MinPriorityQueue();
    pq.enqueue(1)
    pq.enqueue(10)
    pq.enqueue(5)
    console.log(pq.toArray())
NOTE: No need to import or anything

# Repository: 
    https://www.npmjs.com/package/datastructures-js (NPM)
    https://github.com/orgs/datastructures-js/repositories (GitHub)
    Example Documentation: https://github.com/datastructures-js/priority-queue
